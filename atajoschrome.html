Atajos del teclado de Google Chrome

Estos atajos de google chrome van a servirte ya que son combinaciones de teclas que sirven para realizar acciones que posiblemente estemos acostumbrados a hacer con el ratón.

  • Ctrl + N: ABRE UNA NUEVA VENTANA
  • Ctrl + Mayúscula + N:ABRE UNA NUEVA VENTANA EN MODO INCÓGNITO
  • Control + T: ABRE UNA NUEVA VENTANA Y VE A ELLA AUTOMÁTICAMENTE
  • Ctrl + Mayúscula + T: REABRE UNA PESTAÑA RECIÉN CERRADA E IR A ELLA
  • Ctrl + Tab o Ctrl + AvPág: VE A PESTAÑA SIGUIENTE
  • Ctrl + Mayús + Tab o Ctrl + RePág: VE A PESTAÑA ANTERIOR
  • Ctrl + 1 a 8:VE A UNA PESTAÑA ESPECÍFICA
  • Ctrl + 9: VE A ÚLTIMA PESTAÑA
  • Alt + Inicio: ABRE PÁGINA DE INICIO EN PESTAÑA ACTIVA
  • Ctrl + W o Ctrl + F4: CIERRA LA PESTAÑA ACTIVA
