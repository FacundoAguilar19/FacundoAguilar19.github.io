Atajos del teclado de Visual Studio Code

A continuación se presenta una lista con los atajos del teclado más comunes en Visual Studio Code:

  • Ctrl + Shift + P: Abre el cuadro de diálogo de comando.
  • Ctrl + P: Abre el cuadro de diálogo de búsqueda de archivos.
  • Ctrl + Shift + N: Abre una nueva ventana de Visual Studio Code.
  • Ctrl + W: Cierra la ventana activa.
  • Ctrl + Shift + T: Reabre la última pestaña cerrada.
  • Ctrl + Tab: Cambia a la siguiente pestaña.
  • Ctrl + Shift + Tab: Cambia a la pestaña anterior.
  • Ctrl + Shift + F: Abre el cuadro de diálogo de búsqueda en todos los archivos.
  • Ctrl + /: Comenta o descomenta la selección actual.
  • Ctrl + Shift + L: Selecciona todas las apariciones de la selección actual.
